
# HAFIZA OYUNU (Mobil) 

Bu proje ile Android uygulama ve cloud teknolojilerinin kullanılması
amaçlanmaktadır.

Projede belirtilen süre içinde zorluk seviyesine göre kartları 
doğru bir şekilde eşleştirmek gerekmektedir.
